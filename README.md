# medical_expenses_manager

Ruby on Rails application for managing medical expenses deductions

# Commands

### After source update

```
bundle
bundle exec rails assets:precompile RAILS_ENV=[RAILS_ENV]
bundle exec pumactl start -e [RAILS_ENV]
```
